# gestion-eventos-cientificos
Práctica Sesión 1 - Herramientas de gestión de de desarrollo de SW. El proyecto simula el inicio de un desarrollo de software real.
Título del proyecto: Aplicación de gestión de eventos científicos.
Objetivo: Gestionar el registro de asistentes, el cobro de la cuota de inscripción, el alojamiento de los asistentes, pagos a los
conferenciantes, distribución horaria de las charlas, expedición de certificados.

# gestion-eventos-cientificos
Práctica Sesión 1 - Herramientas de gestión de de desarrollo de SW. El proyecto simula el inicio de un desarrollo de software real.
# Título del proyecto: Aplicación de gestión de eventos científicos.

## Objetivo:
Gestionar el registro de asistentes, el cobro de la cuota de inscripción, el alojamiento de los asistentes, pagos a los conferenciantes, distribución horaria de las charlas, expedición de certificados.

## Funcionalidades previstas:
- Generación automática de facturas referentes a la inscripción para aquellas personas que lo soliciten.
- Pago a los ponentes por la impartición de su conferencia mediante transferencia bancaria.
- Gestión de los cobros de las inscripciones.
- Registro de las reservas de hoteles realizadas mediante la mediación de la organización del evento.
